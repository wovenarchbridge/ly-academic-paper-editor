# ly-academic-paper-editor
学术论文智能审校 Skill —— 基于建筑史学者刘妍的论文写作及修改经验生成的AI编辑工具

基于建筑史学者刘妍指导中文学术论文经验的智能审校 Skill，适用于 Kimi 等支持 Skill 机制的 AI 平台。

## 核心功能

| 模块 | 能力 |
|------|------|
| **全局诊断** | 评估研究问题清晰度、分析框架完整性、理论适用性 |
| **逐段精修** | 论点先行、论据跟进、段落过渡、摘要结构优化 |

## 使用方式

在支持 Skill 的 AI 对话平台中导入 `.skill` 文件即可调用。

## 适用场景

- 建筑学（或更广泛的人文社科类）中文学术论文初稿审校
- 期刊投稿前的自我检查
- 研究生论文写作辅导

## 文件说明

- `ly-academic-paper-editor.skill` —— 核心规则库文件（25KB）

## 声明

本 Skill 规则库为个人教学经验总结，仅供学术交流使用。
