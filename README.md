# ly-review-revise-helper.skill
学术论文智能审校 Skill —— 基于建筑史学者刘妍的论文写作、修改与审稿经验生成的AI辅助工具

基于建筑史学者刘妍指导中文学术论文经验的智能审校 Skill，适用于 Kimi 等支持 Skill 机制的 AI 平台。

## 核心功能


| 模块 | 能力 |
|------|------|
| **审稿** | 全局判断 → 审稿人质疑 → AI 替文章辩护（作者分身）→ 出具审稿结论（拒稿/大修/小修/接受）及修改建议 |
| **修改** | 全局诊断 → 结构大修提醒 → 逐段精修 |

## 使用方式

在支持 Skill 的 AI 对话平台中导入 `.skill` 文件即可调用。

## 适用场景

- 建筑学（或更广泛的人文社科类）中文学术论文投稿前自检
- 研究生论文写作辅导与审稿训练
- 审稿人审稿意见参考
- 期刊投稿前的双视角（审稿+修改）交叉检查

## 文件说明

- `ly-academic-paper-editor.skill` —— 核心规则库文件（52KB）

本 Skill 规则库为个人审稿与教学经验总结，仅供学术交流使用。
